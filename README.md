# F-A_translation
F&amp;A is a popular Minecraft mod about scientifically accurate dinosaurs. I'm helping as a volunteer translator to make it more accessible for Turkish children. 

I participate in the translation process through Crowdin, where I localized the mod into my native language, Turkish. This repository showcases my Turkish localization work and tracks my contributions.

I personally enjoy this mod because it presents dinosaurs not as the exaggerated “monsters” often they represented in popular culture, but as real animals that once lived on our planet. This perspective strongly influenced my interest in science.

Original project repository:
https://github.com/TeamFossilsArcheology/FossilsArcheologyRevival
